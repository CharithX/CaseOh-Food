# CaseOh -Food Ordering Application 

 technologies - Next.js Typescript Node.js Express.js  PostgreSQL Prisma

## Installation

Use the Application  

```bash
cd backend
npm install 
```
```bash
cd frontend
npm install 
```
## Run Application

```bash
cd backend
npm run dev
```
```bash
cd frontend
npm run dev
```
Please make sure to update tests as appropriate.
